# PART 1: AUTHORITATIVE SOURCES BY COURSE CATEGORY

## UNIVERSAL PRIMARY SOURCES (Use for ALL Courses)

**Care Quality Commission (CQC)**
- Website: https://www.cqc.org.uk
- Use for: What CQC inspectors check, compliance requirements, fundamental standards
- Key sections: Key Lines of Enquiry (KLOEs), provider guidance, inspection reports
- **Always mention:** "CQC-compliant" or "meets CQC requirements" in first 100 words

**Skills for Health**
- Website: https://www.skillsforhealth.org.uk
- Use for: UK Core Skills Training Framework, learning outcomes, competency standards
- **Most courses align with:** Core Skills Training Framework (CSTF)

**Skills for Care**
- Website: https://www.skillsforcare.org.uk
- Use for: Care Certificate standards, workforce data, best practice
- Key resource: The Care Certificate (15 standards)

**Department of Health and Social Care**
- Website: https://www.gov.uk/government/organisations/department-of-health-and-social-care
- Use for: Care Act 2014, legislation, statutory guidance
